### 代码格式化

- On Windows 　　Shift + Alt + F
- On Mac 　　Shift + Option + F
- On Ubuntu　　 Ctrl + Shift + I

command+shift+P，查找format document，并执行

### 返回上一个光标

- Windows: Alt + ← ;或者 鼠标侧键
- Linux: Ctrl + Alt + - ;貌似数字键盘的减号没效果
- Mac: Ctrl + -

下一个，加上shift

### 搜索文件名

- Mac: Command+P

### 搜索函数名（指定文件范围）

- Mac: Command+P, 输入@func_name

### 搜索函数名（整个项目）

- Mac: Command+P, 输入#func_name
