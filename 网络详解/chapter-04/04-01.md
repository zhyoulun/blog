## 积累的清单

找命令的安装方式：https://command-not-found.com/ip

```bash
# ip
apt install iproute2
# nslookup
apt install dnsutils
# dig
apt install knot-dnsutils
# ping
apt install iputils-ping
# brctl
apt install bridge-utils
# 
apt install curl wget tcpdump telnet lsof

```
